== Installation ==

= AUTOMATIC INSTALLATION =

Automatic installation method is the easiest and most recommended option.

1. Log in to your WordPress Dashboard, navigate to the Plugins menu, and click on ‚ÄúAdd New‚Äù.
2. Enter ‚Äòclick5‚Äô (without quotes) in the search field and click the ‚ÄòSearch Plugins‚Äô button.
3. On the list of search results, click the 'Install Now' button for the 'Sitemap by click5' plugin.
4. Then click the red 'Active' button to finalize the installation.
5. You will be automatically redirected to the plugin Settings page, where you can copy the shortcode and configure both sitemaps - HTML & XML.

= MANUAL INSTALLATION =

1. Download the plugin and unzip the contents. You should see a single folder named 'sitemap-by-click5'.
2. Connect to the FTP server with your WordPress installation.
3. Upload the 'sitemap-by-click5' plugin folder to the /wp-content/plugins/ folder on your FTP server.
4. Continue to step 3 of Automatic Installation instructions above.

== Frequently Asked Questions ==

= Does it support Custom Post Types? =

Yes, our plugin supports Custom Post Types and Custom Taxonomies in HTML and XML sitemaps.

= Does it generate XML Sitemaps? =

Yes, our plugin generates XML Sitemaps, which can be submitted to Google and other search engines to improve your website's visibility.

= Do I need to manually update XML Sitemap? =

No, your XML Sitemap is going to be automatically updated every time there is a change in content on your website.

There are additional settings allowing to use cron job to re-generate XML Sitemap in the background. This option is recommended for websites with more than 1,000 posts and pages.

You can also disable automatic XML Sitemap updates, and do it manually by clicking on the 'Re-generate XML Sitemap' button, whenever you want to update your XML Sitemap.
